# Laboratorio: Arquitectura Web y RDS con Terraform

Despliegue automatizado de una VPC, servidor web (EC2) y base de datos relacional (PostgreSQL en RDS) utilizando prácticas de Infraestructura como Código.

* **Seguridad:** Aislamiento de red (Subredes privadas para RDS) y gestión de secretos con AWS Secrets Manager.
* **Control de Costos:** Uso de instancias t2/t3.micro limitadas a tiempo de ejecución estricto.
